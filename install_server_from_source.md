### 源码安装
#### Zaly源码下载
* gitHub地址
		
		https://github.com/akaxincom/openzaly

* 您可以下载java代码，编译jar包； 也可以选择直接下载打包好的jar包
	* 使用mvn

* 运行jar包[源码安装-jar参数选项](./源码安装-jar参数选项.md)
	
		java -Dsite.port=$PORT -Dhttp.port=$PORT2 -jar openzaly-boot-jar-with-dependencies.jar
